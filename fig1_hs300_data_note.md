# 图1 沪深300基准线数据说明

## 数据来源

图1中的沪深300基准线（黑色虚线）数据来自两个一致的数据源：

1. `data/clean/stock_close_wide.csv` 中的 `沪深300` 列
2. `data/clean/index_300_clean.csv` 中的 `close` 列

**验证结果**：两个数据源的沪深300收盘价完全一致（差异 = 0），数据可靠。

## 归一化方式

- 基准日：2020年第一个交易日（2020-01-02），即数据的第1行（`iloc[0]`）
- 归一化公式：`归一化价格 = 当日收盘价 / 基准日收盘价`
- 基准日归一化值 = 1.0（正确）
