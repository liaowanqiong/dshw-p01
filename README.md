## 3. 宏观经济指标

| 指标 | 选择理由（与股票市场的关联逻辑） |
|------|--------------------------------|
| CPI同比增速（必选） | CPI反映通胀水平，直接影响央行货币政策取向（降息/加息），进而影响市场流动性和股票估值。高通胀通常导致无风险利率上升，压制股票价格；但适度通胀有利于企业盈利增长。 |
| 人民币/美元汇率 | 汇率变化直接影响出口型企业（如比亚迪、长城汽车）的海外营收折算利润，同时影响外资流入A股的意愿，对市场流动性有间接影响。人民币升值利于进口依赖型企业，贬值利好出口企业。 |
| M2同比增速 | M2反映市场流动性总量，M2增速上升意味着"放水"，通常利好股市估值提升。M2与沪深300走势有较强的正相关关系，是预判市场资金面的重要领先指标。 |
| 1年期LPR利率 | LPR是贷款基准利率，直接影响企业融资成本和居民信贷需求。降息利好高负债行业（如能源、物流）和消费类企业；加息则压制估值，尤其是成长型股票。 |
| 工业增加值增速 | 工业增加值反映实体经济活跃度，与制造业（比亚迪、长城汽车、中兴通讯）和能源（中国石油、中国神华）行业景气度密切相关，是基本面驱动股价的核心宏观指标。 |

## 5. 存储方式

### 方式A：CSV（必做）
- 所有原始数据和合并数据以CSV格式存储
- **优点**：通用性强，几乎所有工具均支持；纯文本格式，可直接用Excel打开查看；便于版本控制和差异对比（Git diff）
- **局限性**：不支持列式存储，读取时需加载整个文件；不支持数据类型约束（所有列默认为字符串）；大文件读写速度较慢；不支持压缩，占用空间大

### 方式C：SQLite（进阶，选用）
- 清洗后数据额外存储于 `data/combined/fin_data.db`
- 设计3张表：`stock_price`（个股行情）、`macro_data`（宏观指标）、`stock_info`（股票基本信息）
- **选择理由**：SQLite是轻量级嵌入式数据库，无需安装服务器即可使用SQL查询；支持索引加速查询；支持跨表JOIN，便于多维度分析；适合中等规模数据的结构化管理；数据库文件可随项目一起版本控制
